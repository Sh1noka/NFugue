# NFugue

[![Build status](https://ci.appveyor.com/api/projects/status/u25hrsi0a1d9jecd?svg=true)](https://ci.appveyor.com/project/mchudy/nfugue)

## Getting started

```csharp
using (var player = new Player())
{
```
